1.3 Installation hints for BotE

If fonts are not displayed correctly in BotE, please delete the BotE
font files in your Windows Fonts folder (defaults to: C:\Windows\Fonts)
and replace them with the font files from the "Data\Fonts" folder from
your installation directory. Just copy these font files to your
Windows Fonts folder.

If there are no sounds and you are running BotE in Linux or you can
only use software-sound than please rename the fmodex.dll.linux file to
fmodex.dll. This would replace the fmodex.dll file used in Windows.
Now you should be able to hear sound effects.
